# 🍓 Fruit Slicer Game 🍉

A fun and addictive web-based game where you slice falling fruits with your mouse! Test your reflexes and aim for the highest score before running out of lives.

## 🌐 Live Demo

Play the game online: [https://iam269.github.io/Fruit-Slicer-Game/](https://iam269.github.io/Fruit-Slicer-Game/)

## 🎮 How to Play

- Click "Start Game" to begin
- Move your mouse over the falling fruits to slice them
- Each sliced fruit increases your score
- You have 3 lives (hearts) - miss 3 fruits and it's game over!
- Click "Reset Game" during play to restart

## ✨ Features

- 🎯 Smooth mouse controls for precise slicing
- 🍎 10 different fruit types
- ❤️ Lives system with visual heart indicators
- 🎵 Sound effects when slicing fruits
- 📊 Real-time score tracking
- 🎨 Beautiful wood-themed UI with animations
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## 🚀 Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start slicing fruits!
